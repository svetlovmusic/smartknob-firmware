# Установка новой прошивки Smart Knob Dev Kit

Этот документ описывает полный процесс прошивки устройства с нуля.

## Требования
- компьютер с Windows, macOS или Linux
- кабель USB‑C
- подключение к интернету для скачивания ПО

## 1. Установка Visual Studio Code
1. Перейдите на <https://code.visualstudio.com/>.
2. Скачайте и установите редактор для вашей операционной системы.

## 2. Установка расширения PlatformIO IDE
1. Запустите Visual Studio Code.
2. Откройте меню Extensions (или нажмите `Ctrl+Shift+X`).
3. Введите в поиске `PlatformIO IDE` и нажмите **Install**.

## 3. Получение исходного кода прошивки
1. Если у вас установлен `git`, клонируйте репозиторий командой:
   ```bash
   git clone https://github.com/SeedLabs-it/smartknob-firmware.git
   ```
   Либо скачайте ZIP-архив проекта и распакуйте его.
2. В VSCode выберите **File → Open Folder** и укажите полученную папку `smartknob-firmware`.

## 4. Установка зависимостей
После открытия проекта PlatformIO автоматически загрузит необходимые библиотеки. Дождитесь завершения процесса, о ходе которого VSCode сообщит в нижней панели.

Если PlatformIO не установлен глобально, его можно установить с помощью скрипта `get-platformio.py` в корне репозитория:
```bash
python3 get-platformio.py
```
После завершения установки перезапустите VSCode.

## 5. Подключение устройства
1. Подсоедините Smart Knob к компьютеру через USB‑C.
2. Если устройство не определяется, переведите его в режим загрузчика: удерживайте кнопки **BOOT** и **EN/RST**, затем отпустите **EN/RST**.

## 6. Сборка и прошивка
1. В панели PlatformIO в VSCode раскройте группу **Project Tasks → seedlabs_devkit**.
2. Выполните задачу **Build** для проверки успешной компиляции.
3. Затем выберите **Upload** или **Upload and Monitor**. PlatformIO соберёт проект и запишет прошивку во флеш‑память устройства.
4. При запросе выберите правильный порт (например, `/dev/ttyUSB0` или `COM3`).

## 7. Проверка
После завершения загрузки можно наблюдать вывод устройства через задачу **Monitor** или оставив открытой задачу **Upload and Monitor**. Smart Knob автоматически перезагрузится и будет работать с новой прошивкой.
